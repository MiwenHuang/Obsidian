程序运行时产生的数据都属于临时数据，程序一旦运行结束都会被释放。通过文件可以将数据持久化。

文件类型有两种
1. 文本文件：文件以ASCII码形式存储在计算机中
2. 二进制文件：文件以文本的二进制形似存储在计算机中，用户一般不能直接读懂他们
## 文本文件

### 写文件
```C++
#include <iostream>
#include <fstream>
using namespace std;

void test()
{
    ofstream ofs;
    ofs.open("test.txt", ios::out);
    ofs << "写入数据";
    ofs.close();
}

int main(){
    test();
    return 1;
}
```

### 读文件
读数据有四种方式
```C++
#include <iostream>
#include <fstream>
#include <string>
using namespace std;

void test()
{
    ifstream ifs;
    ifs.open("test.txt", ios::in);
    if (!ifs.is_open())
    {
        cout << "文件打开失败" << endl;
        return;
    }
    // 读数据
    // 第一种
    // char buf[1024]={0};
    // while(ifs>>buf){
    //     cout<<buf<<endl;
    // }
    // 第二种
    // char buf[1024] = {0};
    // while (ifs.getline(buf, sizeof(buf)))
    // {
    //     cout << buf << endl;
    // }

    // 第三种
    string buf;
    while (getline(ifs, buf))
    {
        cout << buf << endl;
    }
    ifs.close();
}

```


## 二进制文件

### 写文件
```C++
class Person
{
public:
    string name;
    int age;
};
void test()
{
    ofstream ofs("person.txt",ios::out|ios::binary);

    Person p={"张三",18};
    ofs.write((const char *)&p,sizeof(Person));
    ofs.close();
}
```

### 读文件
```C++
class Person
{
public:
    string name;
    int age;
};
void test()
{
    ifstream ifs("person.txt", ios::in | ios::binary);
    if (!ifs.is_open())
    {
        cout << "文件打开失败" << endl;
    }
    Person p = {"张三", 18};
    ifs.read((char *)&p, sizeof(Person));
    cout<<p.name<<p.age<<endl;
    ifs.close();
}
```
